Репозиторий-лаборатория, где собраны проекты по проектированию баз данных, оптимизации запросов и разработке отказоустойчивых API. От алгоритмических игр до систем с авторизацией и сложной бизнес-логикой.

---

## 🛠 Глубокая экспертиза (Technical Deep Dive)

В этих проектах я сфокусировался на решении реальных проблем производительности и безопасности:

### 1. 🧬 Проектирование БД и Оптимизация (ORM Excellence)
- **Связи Many-to-Many:** Глубокая проработка таблиц ассоциаций и корректное удаление связанных данных.
- **Решение проблемы N+1:** Использование продвинутых стратегий загрузки SQLAlchemy (`selectinload` для коллекций и `joinedload` для связей "многие-к-одному").
- **Составные Индексы:** Проектирование индексов по нескольким полям для ускорения поиска в высоконагруженных таблицах.
- **Миграции (Alembic):** Управление жизненным циклом БД, автоматическая генерация и ручная правка миграций.

### 2. 🔐 Security & Backend Architecture
- **Auth Flow:** Реализация полного цикла регистрации и входа с использованием JWT-токенов.
- **Crypto:** Хеширование паролей с солью (bcrypt) — пароли никогда не хранятся в открытом виде.
- **Rate Limiting:** Защита эндпоинтов от брутфорса и спам-запросов.
- **Base CRUD Pattern:** Использование универсального базового класса для операций с БД, что сокращает дублирование кода на 40%.

## 📈 Метрики качества кода

Для поддержания чистоты и стабильности в проектах применяются:
- **Type Hinting:** Строгая типизация через `mypy` для минимизации runtime-ошибок.
- **Layered Structure:** Разделение на модели (Models), схемы (Schemas) и сервисы (Services).
- **Asynchronous I/O:** Использование `async/await` для эффективной обработки конкурентных запросов в FastAPI.

## ⚙️ Локальное развертывание (Dev Environment)

1. **Изоляция:** Создайте виртуальное окружение `python -m venv venv`.
2. **Зависимости:** Установите пакеты через `pip install -r requirements.txt`.
3. **Конфигурация:** Настройте параметры БД в `.env` файле (пример в корне).
4. **Запуск:** Используйте `uvicorn main:app --reload` для API-проектов или `python <filename>.py` для скриптов.
